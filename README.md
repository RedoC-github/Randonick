# Randonick
A random-based nickname generator.

In version 0.0.1(alpha), you have to run in like this:

1. Firstly, clone this repository.
```
$ git clone https://github.com/yuanie324/Randonick.git
```

2. Go to cloned folder.
```
$ cd Randonick
```

3. Run.
```
$ gcc randonick.cpp
```
